# Canvas Design Pro - AI绘本版

> 2026-04-14 进化版 · 专为波波AI绘本定制

## 🎯 进化背景

波波AI绘本系列视觉规范：

| 元素 | 规范 |
|------|------|
| 核心IP | 波波 - 头戴蓝色蝴蝶结的小兔子，有缺点、会成长 |
| 尺寸 | 1920×1080（16:9横版）或 1080×1920（9:16竖版）|
| 风格 | 凯迪克奖级别、莫兰迪色系、护眼 |
| 画面要求 | 无文字、纯视觉叙事 |
| 漫画风 | 悬疑漫画风 / 爽文漫画风 |

## 🔥 AI绘本模式（主模式）

### 触发关键词

当用户提及以下任一关键词时，**自动切换到AI绘本模式**：

| 关键词组 | 示例 |
|----------|------|
| IP角色 | 波波、小兔子、蓝色蝴蝶结 |
| 内容类型 | 绘本、故事、配图、封面图 |
| 平台 | 小红书、抖音、绘本 |
| 操作 | 生成图片、作画、画 |

### 波波IP形象规范

```
【基础形象】
- 物种：小兔子
- 特征：头戴蓝色蝴蝶结（波波的标志）
- 性格：有缺点、会成长、温暖治愈
- 年龄：2-6岁儿童视角

【配色规范】
- 主色调：莫兰迪色系（灰粉、灰蓝、灰绿）
- 波波蓝：#6B8E9F（蝴蝶结）
- 背景：柔和渐变，不用纯白
- 禁止：荧光色、高饱和度

【画风要求】
- 凯迪克奖级别：细腻情感、视觉叙事
- 禁止：卡通化、简笔画、流水线
- 建议：油画质感、水彩细腻
```

### 图片尺寸规范

| 用途 | 尺寸 | 比例 | 备注 |
|------|------|------|------|
| 小红书封面 | 1080×1920 | 9:16 | 竖版 |
| 小红书内页 | 1080×1440 | 3:4 | 竖版 |
| 抖音封面 | 1080×1920 | 9:16 | 竖版 |
| 公众号封面 | 900×383 | 2.35:1 | 横版 |
| 通用横版 | 1920×1080 | 16:9 | 横版 |
| 通用竖版 | 1080×1920 | 9:16 | 竖版 |

**重要**：绘本图片**禁止包含文字**，纯视觉叙事。

### 漫画风格规范

#### 悬疑漫画风

```
【特点】
- 光影对比强烈
- 视角低角度仰视
- 色彩偏暗（深蓝、墨紫）
- 悬念感（未知角落、阴影）

【适用场景】
- 黑夜、冒险、未知恐惧
- 波波独自面对挑战

【提示词示例】
cinematic lighting, low angle view, mysterious shadows, deep blue and purple tones, suspenseful atmosphere, children's picture book, Caldecott Award style, soft matte rendering --ar 16:9 --v 6
```

#### 爽文漫画风

```
【特点】
- 色彩明亮温暖
- 视角正面平视
- 高光充足
- 积极向上、阳光灿烂

【适用场景】
- 欢乐时光、友谊、成长
- 波波与朋友互动

【提示词示例】
bright warm lighting, front-facing perspective, golden hour glow, optimistic atmosphere, joyful expression, children's picture book, Caldecott Award style, soft watercolor texture --ar 16:9 --v 6
```

### 构图规范

| 构图 | 适用场景 | 提示词 |
|------|---------|--------|
| 全景 | 展示环境、冒险 | wide shot, establishing scene |
| 中景 | 人物互动、对话 | mid shot, two characters |
| 特写 | 情感表达、细节 | close-up, emotional moment |
| 仰视 | 英雄感、勇敢 | low angle, hero pose |
| 俯视 | 温馨、睡前 | top-down, bird's eye |

### 输出文件

生成后保存到：
```
/tmp/bobo_{故事名}_{YYYYMMDD}_{序号}.png
```

## 🎨 通用视觉设计模式（备用模式）

当用户未提及绘本/波波关键词时，使用原有艺术设计模式。

### 设计哲学流派

| 流派 | 特点 | 示例 |
|------|------|------|
| Brutalist Joy | 粗犷欢快 | 几何色块、醒目 |
| Chromatic Silence | 色场语言 | 几何精确、 Josef Albers风格 |
| Analog Meditation |  analog质感 | 纸纹、留白、日本写真集风格 |
| Organic Systems | 有机生长 | 圆润、群落、模块化 |
| Geometric Silence | 几何沉默 | 网格、精确、Swiss风格 |

### 创作流程

1. **设计哲学创建** → 4-6段文字描述美学方向
2. **推演隐藏线索** → 从用户意图中提炼微妙概念
3. **Canvas创作** → 在画布上表达为PNG/PDF
4. **精修** → 二次打磨，追求博物馆级品质

### 质量标准

**必须达到**：
- 看起来像数千小时的匠心之作
- 专家级执行：排版、间距、色彩无可挑剔
- 元素不重叠、边界清晰、呼吸感充足
- 文字极简（仅作为视觉元素）
