# 财姐 Stock Monitor Pro

> v2.0 财姐进化版 | 数据源：东方财富（主）+ 腾讯财经（辅）| 底线：9条内置

---

## ⚠️ 底线内置（所有预警以此为准）

| 底线 | 阈值 | 动作 |
|------|------|------|
| 止损 | -3% | 无条件触发告警 |
| 止盈 | +5% | 分批出（半仓留博二波） |
| 超短线 | 10:30前 | 强制出，时间到即走 |
| 单票仓位 | ≤15% | 告警超仓 |
| 不推板块 | 科创/创业/北交所 | 监控时自动过滤 |
| 涨跌比 | <0.3 | 建议不开新仓 |
| 连亏熔断 | 2天/-50% 3天/清仓 | 连续触发时自动告警 |

## 7大预警规则（财姐版阈值）

| # | 规则 | 触发条件 | 权重 | 状态 |
|---|------|----------|------|------|
| 1 | **成本百分比** | 盈利+5% / 亏损-3% | ⭐⭐⭐ | ✅阈值校准 |
| 2 | 日内涨跌幅 | 个股±4% / ETF±2% | ⭐⭐ | ✅不变 |
| 3 | 成交量异动 | 放量>2倍5日均量 / 缩量<0.5倍 | ⭐⭐ | ✅不变 |
| 4 | 均线金叉/死叉 | MA5上穿/下穿MA10 | ⭐⭐⭐ | ✅不变 |
| 5 | RSI超买超卖 | RSI>70超买 / RSI<30超卖 | ⭐⭐ | ✅不变 |
| 6 | 跳空缺口 | 向上/向下跳空>1% | ⭐⭐ | ✅不变 |
| 7 | 动态止盈 | 盈利+5%后回撤3%减半，回撤5%清仓 | ⭐⭐⭐ | ✅财姐化 |

**🆕 第8条 超短线时间锁（财姐新增）**
- 超短线持仓：次日10:30前未自动触发卖出提醒
- 告警格式：「时间到，强制执行」

**🆕 第9条 涨跌比风控（财姐新增）**
- 涨跌比<0.3时：发出"大盘极差，不开新仓"告警

**🆕 第10条 连亏熔断（财姐新增）**
- 连亏2天：告警"减仓50%"
- 连亏3天：告警"清仓"

## 分级预警系统

| 级别 | 触发条件 | 颜色 | 行动 |
|------|----------|------|------|
| 🚨 紧急 | 多条件共振（3项+） | 🔴红 | 立即处理 |
| ⚠️ 警告 | 2个条件触发 | 🟡黄 | 当日处理 |
| 📢 提醒 | 单一条件触发 | 🔵蓝 | 关注 |

**颜色习惯：** 🔴红=涨/盈利 | 🟢绿=跌/亏损（中国习惯）

## 监控配置格式

```python
{
    "code": "002240",       # 深市主板 sz，沪市 sh
    "name": "盛新锂能",
    "market": "sz",
    "cost": 41.969,         # 持仓成本
    "shares": 300,           # 持仓数量
    "type": "individual",   # 个股/etf/gold
    "alerts": {
        "cost_pct_above": 5.0,     # 止盈+5%（财姐标准）
        "cost_pct_below": -3.0,    # 止损-3%（财姐标准）
        "change_pct_above": 4.0,
        "change_pct_below": -4.0,
        "volume_surge": 2.0,
        "ma_monitor": True,
        "rsi_monitor": True,
        "gap_monitor": True,
        "trailing_stop": True,       # 动态止盈：+5%后回撤3%减半
        "daytrade_expiry": "10:30",  # 🆕超短线时间锁
        "stop_loss_hard": -3.0       # 🆕硬止损
    }
}
```

**数据来源：** 东方财富 caifuhao.eastmoney.com（主）+ 腾讯财经 qt.gtimg.cn（辅）
